왜 이 원데이인가요?

최신 GitHub 기술을 직접 체험하고 실전 산출물까지 가져가는 6시간

🎯

최신 GitHub Universe 2025 브리핑

Agent HQ(오케스트레이션), 채널별 Copilot 할당, VS Code Plan Mode, AGENTS.md, MCP, Code Quality/메트릭스 핵심만 간결하게. 일부 기능은 실습 산출물 제작에 곧바로 활용.

🎸

바이브코딩 '즉시 체감'

GitHub Spark에서 요구사항 입력 → 미니앱 생성 → 미리보기 → 원클릭 Publish까지 실습. VS Code/코드스페이스 연동으로 기능 고도화.

📈

실전 산출물: AEO/GEO 블로그

AEO(Answer Engine Optimization), GEO(Generative Engine Optimization) 개념 적용. JSON‑LD(Article/FAQPage) 삽입 + 자동배포(GitHub Pages/Azure SWA) 경험.

🎬

공식 레퍼런스

GitHub Universe 2025 Keynote (YouTube)를 참조하여 최신 내용을 반영합니다.
