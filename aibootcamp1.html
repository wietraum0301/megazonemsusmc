AI Foundry 기초 과정 (2일)

모델 탐색부터 배포, 관리까지 Azure AI Foundry의 핵심 전 과정을 마스터합니다.

01

1일차: 기초 및 모델 탐색

Step 1: Azure AI Foundry 개요

통합 AI 플랫폼 아키텍처 이해 및 리소스 환경 구축

Azure 구독 연결, AI Hub·Project 구조 파악, 역할 기반 접근 제어(RBAC) 설정까지 실습하며 엔터프라이즈 환경의 기반을 완성합니다.

Step 2: 모델 카탈로그 활용

GPT-4, Phi-3 등 주요 LLM/SLM 모델 탐색 및 성능 벤치마킹

Foundry 내 모델 카탈로그에서 200개 이상의 모델을 비교하고, 태스크 유형별(생성·분류·요약) 최적 모델 선택 기준을 직접 실험합니다.

Step 3: AI Studio 프롬프트 설계

시스템 메시지 최적화 및 구조적 프롬프트 엔지니어링 실습

Few-shot 예시, Chain-of-Thought, 역할 지정 기법을 활용한 시스템 프롬프트를 작성하고 응답 품질 변화를 비교·분석합니다.

Step 4: AI 서비스 보안 설정

엔터프라이즈 환경을 위한 보안 및 Content Safety 적용

Azure Content Safety API를 연동해 유해 콘텐츠 필터링 정책을 구성하고, Private Endpoint와 VNet 통합으로 네트워크 격리 환경을 구축합니다.

02

2일차: 고급 오케스트레이션 및 배포

Step 5: RAG 아키텍처 구현

Azure AI Search 연동 및 고유 데이터를 활용한 지능형 검색 구축

사내 문서를 벡터 임베딩으로 인덱싱하고, 하이브리드 검색(키워드+시맨틱)으로 정확도 높은 컨텍스트를 LLM에 주입하는 파이프라인을 구성합니다.

Step 6: Prompt Flow 워크플로우 설계

LLM 애플리케이션의 복잡한 로직을 시각적으로 설계 및 테스트

분기·루프·병렬 처리가 가능한 DAG 기반 플로우를 노코드 UI로 설계하고, Python 노드를 삽입해 비즈니스 로직을 확장하는 방법을 실습합니다.

Step 7: 모델 평가 및 고도화

답변 품질 평가 지표 설정 및 자동화된 테스트 환경 구축

Groundedness, Coherence, Relevance 등 AI 품질 메트릭을 설정하고, 골든 데이터셋 기반의 배치 평가 파이프라인을 자동화합니다.

Step 8: 실무 배포 및 모니터링

엔드포인트 생성, 실시간 트래픽 관리 및 토큰 사용량 최적화

관리형 온라인 엔드포인트에 모델을 배포하고, Azure Monitor 대시보드로 레이턴시·토큰 비용·오류율을 실시간 모니터링하는 운영 체계를 완성합니다.

바이브 코딩 실전 워크숍 (1일)

GitHub Copilot을 파트너 삼아 아이디어를 실제 프로덕트로 만드는 차세대 개발 경험

lightbulb Session 01

바이브 코딩 마인드셋 & 기획

AI 시대의 개발 생산성 이해 및 구현할 프로젝트 아이디어 구체화

전통적 개발 사이클과 AI 협업 개발의 차이를 비교하고, 1일 내 완성 가능한 범위의 프로젝트 스코프를 정의합니다.

description Session 02

AI용 PRD 및 아키텍처 설계

GitHub Copilot이 가장 잘 이해할 수 있는 요구사항 정의서와 구조 설계

Copilot이 코드를 잘 생성하도록 맥락이 풍부한 PRD를 작성하고, 컴포넌트 구조와 데이터 흐름을 다이어그램으로 시각화합니다.

terminal Session 03

실전 코딩: 프론트엔드 및 로직

Copilot Chat과 대화를 통해 핵심 기능과 인터페이스를 고속 구현

인라인 제안과 Chat 패널을 함께 활용해 UI 컴포넌트와 상태 관리 로직을 대화 흐름 속에서 빠르게 완성합니다.

api Session 04

API 연동 및 데이터 통합

외부 API 연동 및 백엔드 로직 구성을 통한 서비스의 완성도 향상

REST·GraphQL API 호출 코드를 Copilot이 자동 생성하게 하고, 오류 처리·인증 흐름까지 프롬프트 한 줄로 완성하는 패턴을 익힙니다.

bug_report Session 05

디버깅 및 코드 최적화

AI를 활용한 오류 수정, 리팩토링 및 성능 개선 실습

스택 트레이스를 붙여넣어 원인 분석과 수정 코드를 동시에 얻고, 중복 코드 제거·함수 분리·타입 안정성 강화를 Copilot과 함께 진행합니다.

rocket_launch Session 06

성과 공유 및 워크플로우 리뷰

최종 결과물 시연 및 개인별 AI 협업 워크플로우 회고

팀원들 앞에서 구현 결과를 라이브 데모하고, 각자가 발견한 Copilot 활용 팁과 워크플로우를 공유해 학습 내용을 내재화합니다.

hub

AI Foundry 기초 과정

기업용 AI 서비스의 핵심인 Azure AI Foundry의 모든 스택을 2일간 집중적으로 학습합니다. 구축 → 평가 → 배포의 전 사이클을 직접 경험합니다.

terminal

실무 지향 오케스트레이션

단순 API 호출을 넘어 Prompt Flow를 통한 고도화된 AI 로직 제어 역량을 확보합니다. 복잡한 비즈니스 로직도 시각적으로 설계·관리합니다.

code_blocks

바이브 코딩 실전 워크숍

기존 개발 방식을 탈피하여 1일 동안 AI와 대화하며 프로덕트를 완성하는 실전 경험을 가집니다. 아이디어를 즉시 코드로 전환하는 감각을 키웁니다.
