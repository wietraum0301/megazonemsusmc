Agentic AI 시대를 준비하기 위한
AI 전문 교육 프로그램

개발자와 아키텍트를 위한 실무 중심 교육. Azure AI Foundry 기초 과정과 GitHub Copilot 실전 워크숍 중 목적에 맞는 프로그램을 선택하세요.

AI Foundry 기초 과정 (2일)

모델 탐색부터 배포, 관리까지 Azure AI Foundry의 핵심 전 과정을 마스터합니다.

01

1일차: 기초 및 모델 탐색

Step 1: Azure AI Foundry 개요

통합 AI 플랫폼 아키텍처 이해 및 리소스 환경 구축

Step 2: 모델 카탈로그 활용

GPT-4, Phi-3 등 주요 LLM/SLM 모델 탐색 및 성능 벤치마킹

Step 3: AI Studio 프롬프트 설계

시스템 메시지 최적화 및 구조적 프롬프트 엔지니어링 실습

Step 4: AI 서비스 보안 설정

엔터프라이즈 환경을 위한 보안 및 Content Safety 적용

02

2일차: 고급 오케스트레이션 및 배포

Step 5: RAG 아키텍처 구현

Azure AI Search 연동 및 고유 데이터를 활용한 지능형 검색 구축

Step 6: Prompt Flow 워크플로우 설계

LLM 애플리케이션의 복잡한 로직을 시각적으로 설계 및 테스트

Step 7: 모델 평가 및 고도화

답변 품질 평가 지표 설정 및 자동화된 테스트 환경 구축

Step 8: 실무 배포 및 모니터링

엔드포인트 생성, 실시간 트래픽 관리 및 토큰 사용량 최적화

바이브 코딩 실전 워크숍 (1일)

GitHub Copilot을 파트너 삼아 아이디어를 실제 프로덕트로 만드는 차세대 개발 경험

lightbulb Session 01

바이브 코딩 마인드셋 & 기획

AI 시대의 개발 생산성 이해 및 구현할 프로젝트 아이디어 구체화

description Session 02

AI용 PRD 및 아키텍처 설계

GitHub Copilot이 가장 잘 이해할 수 있는 요구사항 정의서와 구조 설계

terminal Session 03

실전 코딩: 프론트엔드 및 로직

Copilot Chat과 대화를 통해 핵심 기능과 인터페이스를 고속 구현

api Session 04

API 연동 및 데이터 통합

외부 API 연동 및 백엔드 로직 구성을 통한 서비스의 완성도 향상

bug_report Session 05

디버깅 및 코드 최적화

AI를 활용한 오류 수정, 리팩토링 및 성능 개선 실습

rocket_launch Session 06

성과 공유 및 워크플로우 리뷰

최종 결과물 시연 및 개인별 AI 협업 워크플로우 회고

hub

AI Foundry 기초 과정

기업용 AI 서비스의 핵심인 Azure AI Foundry의 모든 스택을 2일간 집중적으로 학습합니다.

terminal

실무 지향 오케스트레이션

단순 API 호출을 넘어 Prompt Flow를 통한 고도화된 AI 로직 제어 역량을 확보합니다.

code_blocks

바이브 코딩 실전 워크숍

기존 개발 방식을 탈피하여 1일 동안 AI와 대화하며 프로덕트를 완성하는 실전 경험을 가집니다.
