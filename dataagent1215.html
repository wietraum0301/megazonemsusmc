Microsoft Fabric Data Agent 실무 교육

자연어 질의로 데이터를 분석하는 AI 에이전트 구축 실전

📅 2025년 12월 15일(월) 09:00~17:00 · 점심시간 제공

Microsoft Fabric의 Data Agent 기능을 활용하여 데이터 접근성과 분석 효율을 높이는 방법을 학습합니다.
실무에 곧바로 적용 가능한 자연어 질의(Natural Language Query) 기반 데이터 분석 환경을 구축하고,
Azure AI Foundry/Copilot Studio와 연계하여 조직 맞춤형 AI 에이전트를 완성하는 과정입니다.

교육 목표

🎯 Fabric & Data Agent 이해

Microsoft Fabric과 Data Agent의 구조 및 역할을 명확히 이해합니다.

🔗 데이터 소스 연동

Data Agent 생성 및 데이터 소스 연동 실습을 수행합니다.

💬 자연어 질의 구현

Natural Language Query의 내부 처리 방식을 이해하고 에이전트 응답 품질을 향상시킵니다.

🤖 AI 에이전트 구축

Azure AI Foundry(Copilot Studio)와 연계하여 AI 에이전트를 구축합니다.

📊 셀프서비스 BI 환경

조직 내 데이터 분석 자동화 및 셀프서비스 BI 환경을 구현하는 역량을 강화합니다.

• Power BI, SQL 기반 분석 경험이 있고 자연어 기반 데이터 질의 운영을 도입하고자 하는 인원

• Microsoft Fabric 환경에서 AI 에이전트를 구축하거나 자동화 에이전트를 개발하고자 하는 담당자

• 조직 내 BI 시스템의 대화형 AI 기능을 도입하고자 하는 담당자

1H
1. Fabric 및 Data Agent 개요

Microsoft Fabric의 구성 요소와 Data Agent의 개념 및 활용 목적을 이해합니다.

  • Microsoft Fabric 구성요소 이해
  • Data Agent의 개념 및 활용 목적
  • 주요 기능과 한계점
  • 적용 시나리오 소개
  • 권장 및 라이선스 설명
2H
2. Fabric 환경 구성 및 데이터 준비

다양한 데이터 소스를 생성하고 질문용 예제 데이터를 준비합니다.

  • 데이터소스 생성 (Lakehouse, Warehouse, Power BI, KQL DB)
  • 데이터 등록
  • 질문용 예제 데이터 준비
3H
3. Data Agent 생성 및 구성

Data Agent를 생성하고 지침 작성, 예제 질의 등록 등 전체 프로세스를 실습합니다.

  • 새 Data Agent 생성 절차
  • 데이터소스 추가 및 데이터 선택
  • 지침(Instructions) 작성
  • 예제 질의(Example Queries) 등록
  • 테스트 및 결과 확인
  • 에이전트 게시 및 공유
3H
4. AI Foundry 및 Copilot Studio에서 Data Agent 활용

Azure AI Foundry와 Copilot Studio에서 Data Agent를 연결하고 Teams에 배포합니다.

  • Copilot Studio/Azure AI Foundry 소개
  • Data Agent 연결 및 연동 설정
  • 트리거 묶기 및 시나리오 구성
  • Teams 팀 및 워칭된 채널 배포
