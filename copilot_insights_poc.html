Copilot for Fabric 대화형 인사이트 대시보드 PoC

AI 기반 대시보드 생성과 자연어 질문으로 얻는 즉각적인 통찰

PoC 개요

본 PoC는 건설 설계 컨설팅 엔지니어링 및 유사 산업을 대상으로 합니다. Microsoft Fabric의 Copilot 기능을 활용하여 프로젝트·매출 데이터를 자동으로 시각화하고, 대화형 인사이트를 제공하는 대시보드를 구축합니다. Copilot은 Real‑Time Hub 또는 KQL Queryset에서 데이터 테이블을 선택하기만 하면 자동으로 대시보드를 생성하며, 인사이트 페이지와 데이터 프로필 페이지를 함께 제공하므로 특별한 기술 없이도 빠르게 결과를 얻을 수 있습니다【601077512807611†L308-L312】. 또한 Power BI Q&A 기능과 Data Agent를 통해 “이번 달 매출 추세”와 같은 자연어 질문에 대해 카드, 차트 등 시각적 답변을 자동으로 생성해 줍니다【221241802478673†L531-L547】.

데이터 통합 & 모델링

  • 설계 도면, 일정, 비용, 매출, 리소스 데이터를 OneLake에 통합하여 사일로를 제거
  • 스타 스키마 모델링과 DAX를 사용해 공정률, 비용 변동, 매출 추이, 리소스 사용률을 계산
  • 월별·연도별 매출 트렌드와 변동률을 시각화해 경영 판단을 지원【221241802478673†L460-L469】
  • Power BI Home 및 Reports에서 대시보드와 보고서를 체계적으로 관리
  • 설계 도면 자체는 집계되는 팩트가 아니므로 스타 스키마의 팩트·차원 원칙에 따라 도면 번호, 버전, 담당자 등의 메타데이터를 차원 테이블로 관리하고【409545552018736†L439-L454】, 3D 모델은 Autodesk Connector나 Vcad 같은 전용 시각화로 보고서에 포함할 수 있도록 연계

  • Copilot의 Q&A 기능을 통해 보고서 페이지에 대한 요약 및 DAX 제안 등 AI 기능 활용

PoC 진행 절차

1. 요구 사항 분석

프로젝트 및 경영 지표(KPI), BIM/ERP 시스템, 매출·비용 데이터 등 데이터 소스를 파악하고 PoC 목표를 정의합니다.

2. 데이터 수집 및 모델링

OneLake에 여러 시스템을 연결해 데이터를 수집하고, 스타 스키마 모델로 변환하여 분석 기반을 마련합니다.

3. 대시보드 & Q&A 구축

Copilot을 이용하여 대시보드를 자동 생성하고, Power BI Q&A와 Data Agent를 통해 자연어 질문으로 인사이트를 탐색하는 기능을 구현합니다.

4. 데모 및 평가

완성된 대시보드와 Q&A 기능을 시연하여 실질적인 효과를 평가하고, 본격적인 도입을 위한 로드맵을 제시합니다.

신청 대상

본 PoC는 건설 설계 컨설팅 엔지니어링 분야의 경영진, 프로젝트 관리자, 설계자, 데이터 분석가 등 프로젝트·매출 데이터를 통해 의사결정을 향상시키려는 모든 분들을 대상으로 합니다. 데이터 기반 설계·공정 관리와 매출 추이 분석을 동시에 경험하고자 하는 조직의 참여를 환영합니다.
