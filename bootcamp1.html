🚀 바이브 코딩 특징
AI Foundry 기초 과정에서 체득한 지식을 활용해 풀스택 AI 애플리케이션을 실제로 제작해보는 실전 중심의 프로그램입니다. 다양한 바이브 코딩 툴(MCP, PRD 작성 툴 등)을 선수 학습하고, 나머지 시간은 실제 제품을 만들어보며 높은 만족도를 제공합니다.
🎨 바이브 코딩 실전 워크숍
1
오리엔테이션
• Vibe Coding 소개
• 목표설정: 나만의 AI 앱을 기획 → 프롬프트 설계 → API 연동 → 제작 → 발표
• 진행 방식 및 시간 안내
2
아이디어 발굴
• 아이디어 도출 가이드
🛠️ 실습: 각자 최소 3개 아이디어 작성 → 1개 최종 선택
3
PRD 작성
• PRD 기본 구조 설명
• PRD 작성 노하우
🛠️ 실습: 각자 PRD 문서 작성 (AI 도구 활용)
4
프롬프트 설계
• 파인튜닝 없이 프롬프트만으로 앱 기능 구현
• 기능 단위로 “요청 프롬프트”를 설계하는 법
🛠️ 실습: AI에게 API 코드(요청/응답) 예시 생성시켜 보기
5
API 연동 & 뼈대 앱 제작
• Azure OpenAI API 개요 및 키 발급
🛠️ 실습: 간단한 UI에서 API 동작 확인
6
자유 개발 시간
• 각자 앱 고도화
• 강사 개별 멘토링
🛠️ 목표: 발표 가능한 수준의 동작하는 앱
7
개인 발표 & 공유
• 1인당 7~10분 발표
• 동료 피드백 (간단한 질문/격려)

마무리
