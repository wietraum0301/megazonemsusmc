Power BI & AI 매출 분석 교육

매출 데이터 경영지표 시각화 + Fabric / Copilot Data Agent 실습

교육 개요

본 교육은 기업의 매출 데이터를 기반으로 경영 지표를 시각화하고, Microsoft Fabric과 Copilot을 활용한 AI 기반 Data Agent를 체험하는 핸즈온 워크숍입니다. 데이터 통합부터 대시보드 설계, AI 도우미 구축까지 최신 기능을 경험합니다.

핵심 세션

Power BI 매출 시각화 (3/4)

  • 중앙 데이터 저장소에서 매출·재고 데이터 수집
  • 스타 스키마 모델링과 DAX 기반 핵심 지표 작성
  • 고객·제품별 매출 분석, 성장률과 재고 회전율 시각화
  • 세그먼트 분석, KPI 경보 및 고급 시각화 기법

Fabric + Copilot Data Agent (1/4)

  • OneLake를 통한 데이터 통합 및 사일로 제거
  • Fabric Data Agent 개요와 활용 사례
  • Copilot Studio에서 AI Agent 생성 및 트리거 설정
  • 실시간 Q&A와 멀티 에이전트 협업 체험

참가 대상 및 혜택

본 교육은 기업 경영진, 데이터 분석가, IT 담당자 등 매출 데이터를 활용하여 경영 의사결정을 개선하고자 하는 모든 분들을 대상으로 합니다. 교육을 통해 다음을 얻을 수 있습니다.

  • 실제 기업 매출 데이터를 활용한 Power BI 모델링과 시각화 역량
  • Fabric OneLake를 기반으로 한 데이터 통합 전략
  • Copilot과 Data Agent를 통한 AI 대화형 분석 경험
  • 미래 AI 서비스와 통합된 데이터 기반 의사결정 역량 강화
