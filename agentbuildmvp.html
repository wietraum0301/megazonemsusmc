7단계로 완성하는 AI 에이전트

입문자도 OK! Python 기초부터 프로덕션까지 한번에

CH 1
첫걸음: "안녕, AI야!" Python으로 시작하기
AI와 처음 대화해보세요. Python으로 첫 AI 앱을 만들고, 내가 원하는 대로 움직이게 만드는 법을 배웁니다.
  • Azure OpenAI 서비스 세팅 완료!
  • Python으로 AI와 대화하기
  • AI한테 잘 부탁하는 법 (프롬프트 엔지니어링)
  • 나만의 챗봇 뚝딱 만들기
CH 3
AI 에이전트가 뭐길래? 기초부터 탄탄하게
AI가 혼자 생각하고 행동한다고? 어떻게 가능한지 알아보고, 직접 똑똑한 AI 에이전트를 만들어봅니다.
  • AI 에이전트의 뇌 구조 이해하기
  • 생각하는 AI 만들기 (ReAct, Chain-of-Thought)
  • AI한테 도구 사용법 가르치기
  • AI가 기억하게 만들기
CH 4
프로들이 쓰는 AI 설계 비법 대공개
실무에서 검증된 AI 에이전트 패턴들을 배우고, 내 프로젝트에 바로 적용해봅니다.
  • 정보를 찾아주는 똑똒이 AI (RAG)
  • 계획 세우고 실행하는 AI
  • 스스로 반성하는 AI (Self-Reflection)
  • 도구를 효율적으로 쓰는 법
CH 5
AI들의 드림팀 만들기: 협업의 기술
AI 한 명보다 여러 명이 낫죠! 각자 역할을 나눠서 협력하는 AI 팀을 만들어봅니다.
  • AI 팀 구조 설계하기
  • AI들끼리 대화하게 만들기
  • 역할별 전문 AI 배치하기
  • 팀워크 최적화의 기술
CH 6
무한 확장의 비밀: MCP 마스터하기
작게 시작해서 크게 키우는 법! 더 많은 기능을 쉽게 추가할 수 있는 확장형 AI를 만듭니다.
  • MCP가 뭐길래? 쉽게 이해하기
  • 확장 가능한 AI 서버 만들기
  • AI 기억력 10배 높이기
  • 커질수록 강해지는 구조 설계

교육 전 필수 준비 사항

교육 시작 전 아래 항목을 반드시 준비해 주세요

☁️ Azure 리소스 준비

  • Microsoft Azure 계정 및 Azure OpenAI 서비스 리소스
  • GPT-4.1 모델 배포
  • text-embedding-3-small 모델 배포
  • Document Intelligence 배포
  • AI Search 배포

💻 개발 환경 설정

  • Python 3.11.9 설치 (필수) — 다운로드
  • VS Code 설치 — 다운로드

📦 Python 패키지 설치

pip install -r requirements.txt

⚙️ 환경 변수 설정 (.env 파일)

프로젝트 루트에 .env 파일을 생성하고 아래 내용을 입력하세요:

AZURE_OPENAI_API_VERSION      = '2024-12-01-preview'
AZURE_OPENAI_DEPLOYMENT_NAME  = "gpt-4.1"
AZURE_OPENAI_API_KEY          = ""
AZURE_OPENAI_ENDPOINT         = ""
AZURE_TEXT_EMBEDDING_MODEL    = "text-embedding-3-small"
AZURE_DOCUMENT_INTELLIGENCE_ENDPOINT = ""
AZURE_DOCUMENT_INTELLIGENCE_KEY      = ""
AZURE_SEARCH_ENDPOINT         = ""
AZURE_SEARCH_API_KEY          = ""
AZURE_SEARCH_INDEX_NAME       = "company-policy-index"
